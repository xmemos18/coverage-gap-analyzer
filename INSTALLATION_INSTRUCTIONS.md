# Installation Instructions for New Features

## Required Package Installations

Run the following command to install PDF generation and charting libraries:

```bash
npm install @react-pdf/renderer recharts lucide-react
```

Or with yarn:

```bash
yarn add @react-pdf/renderer recharts lucide-react
```

## Package Purposes

- **@react-pdf/renderer**: Generate downloadable PDF reports of insurance recommendations
- **recharts**: Create interactive charts for cost comparisons and visualizations
- **lucide-react**: Icon library for info tooltips and UI enhancements

## After Installation

All components have been created and are ready to use once packages are installed.
The app will automatically use these features without additional configuration.
